Welcome to Rent Management Application

This application is created as a modern way to manage apartment rental.

The base object is an Apartment. Every apartment can be rented in non-intersecting periods of time. Tenants and payments can be assigned to each rent. Tasks are related directly to the apartment. And in the end, there is a payment track feature.

All of it is still in development stage.
